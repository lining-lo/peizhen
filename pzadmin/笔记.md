#### 1. vite 初始化项目

```
1 创建项目：npm create vite@latest my-vue-app -- --template vue
2 下载依赖：nmp install
3 启动项目；npm run dev
```

#### 2. 路由配置

```
1 安装依赖：npm install vue-router@4
2 创建路由组件（views）
3 配置路由（router）
4 挂载到 vue 实例（app.use）
5 指定位置展示（RouterView>）
```
